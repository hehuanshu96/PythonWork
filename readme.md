# Python Work（Python3.6）
## dailycoding
&emsp;&emsp;dailycoding中都是一些日常小编程，其中每个文件夹之间相互独立。
各自包含必要的代码、数据文件和运行结果（如果存在的话）。
### PIL
一个创建验证码图片的脚本。
### TextStatistics
统计天龙八部中人物名字出现的次数，并绘制直方图显示统计结果。
### commendLineParsing
接收并解析命令行信息。
### email_in_diffway
邮件收发
### python_UI
一个用于浏览和标记图片的GUI。在机器学习中，它可以帮你来标记样本，标记结果保存在~/statistcs.csv中。
